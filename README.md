repost.

## Dependencies

### LaTeX

- latex packages
- Pygments

##### LaTeX Packages
```
# arch-based system
sudo pacman -S texlive-most texlive-lang

# debian-based system
sudo apt-get install texlive-full
```

##### Pygments
```
# arch-based system
sudo pacman -S python-pip

# debian-based system
sudo apt install python-pip

# any systems
pip install Pygments
```
